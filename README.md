Student Expense Tracker (Java Project)
Project Description

The Student Expense Tracker is a Java-based console application developed to help students manage and monitor their daily expenses in a simple and efficient way. Many students spend money daily on food, travel, mobile recharge, and other activities but do not keep track of where their money is going. This project solves that real-world problem by providing a simple system to record and analyze expenses.

Features
Add daily expenses
View all expenses
Calculate total expenses
Show highest expense
Simple and user-friendly menu system
Technologies Used
Java (Core Java)
ArrayList
OOP Concepts
Scanner Class
VS Code
How to Run the Project
Download the project
Open the folder in VS Code
Open Main.java
Run the program
Follow the menu options shown on the screen
